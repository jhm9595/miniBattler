# Cosmic Archive

A hexagonal inventory puzzle + asynchronous PvP auto-battler built with React, PixiJS, and Tailwind CSS.

## Features
- **Hexagonal Grid System**: Axial (q, r) coordinate system for precise item placement.
- **Dynamic Rotation**: 6-direction rotation using matrices.
- **Cosmic Synergy Engine**: Visualized with neon synergy links between resonant fragments.
- **Fusion & Synthesis**: Secret recipes and 3-merge evolution logic.
- **Premium UI**: Dark-themed, cosmic aesthetic with glassmorphism and animations.

## Tech Stack
- **Frontend**: React 18
- **Rendering**: PixiJS 7 + @pixi/react
- **Style**: Tailwind CSS + Framer Motion
- **State**: Zustand

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Gameplay
- **Shop**: Buy cosmic fragments for 5 Gold each.
- **Inventory**: Click fragments to deploy them into the Archive Grid.
- **Archive**: Drag particles to arrange them. Press **[R]** while dragging to rotate.
- **Resonance**: Keep matching items or follow secret recipes for massive power spikes.
